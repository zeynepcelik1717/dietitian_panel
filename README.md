# Diyetisyen Yönetim Paneli 🥗

Diyetisyenlerin danışanlarını yönetebileceği basit ve etkili bir web uygulaması.

## Özellikler (Planlanmış)
- 🔐 Diyetisyen giriş/kayıt sistemi
- 👥 Danışan ekleme, listeleme, düzenleme
- 📊 Kilo takibi ve grafiksel görselleştirme
- 🍽️ Beslenme planı oluşturma
- 🤖 (v2) LLM destekli danışan chatbot

## Stack
- **Frontend + Backend:** Streamlit
- **Database:** Supabase (PostgreSQL)
- **Auth:** Supabase Auth
- **Visualization:** Plotly

## Kurulum
\`\`\`bash
git clone https://github.com/zeynepcelik1717/dietitian-panel.git
cd dietitian-panel
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
\`\`\`
